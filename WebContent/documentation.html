    Get List of books(HTTP GET)

    A GET method to intense-headland-8238.herokuapp.com/rest/books can be made to return JSON representations of List of books.

    A successful request will result in an HTTP status code of 200 and a JSON Array list of books in the response body.

    Get List of Imagenames(HTTP GET)

    To get list of imagenames, request an HTTP GET to this URL intense-headland-8238.herokuapp.com/rest/books/images

    For example, intense-headland-8238.herokuapp.com/rest/books/images will return:

    JSON

    [
{"imagename":"ds.jpg"},
{"imagename":"java.jpg"},
{"imagename":"os.jpg"},
{"imagename":"cn.jpg"},

]

    Get Image(HTTP GET)

    To get the image of a book,request an HTTP GET to this URL intense-headland-8238.herokuapp.com/rest/books/files/ds.jpg

    A successful request will result in an HTTP status code of 200 and an image of a book in the response body.

    JSON

    [
    displays the picture of the book.
